# Welcome!

This C++ template lets you get started quickly with a simple one-page playground.

```C++ runnable
#include <iostream>

using namespace std;

int main()
{
     int a[54] = {};
 
     int b[54] = {};
 
     int* x = a;
 
    int* const y = a;
 
    b = x;
 
    b = y;
 
    return 0;
 } 
```
